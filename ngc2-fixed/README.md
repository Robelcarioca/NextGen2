# 🚀 Next-Gen Creative — Full Website System

## 📁 Files
- `index.html` — Customer-facing portfolio website
- `admin/index.html` — Admin panel (Login: admin / nextgen2025)
- `site-config.json` — ALL website content (hero, services, testimonials, about, pricing, contact)
- `designs.json` — Portfolio designs with images

## 🌐 Deploy to Vercel
1. Go to vercel.com → New Project
2. Drag & drop this entire folder
3. Click Deploy → live in 30 seconds!

## 🔑 Admin Panel
URL: yoursite.vercel.app/admin
Login: admin / nextgen2025

## 🎨 How to Update Your Live Site
1. Login to /admin
2. Make changes (designs, pricing, reviews, about, etc.)
3. Click "Export & Deploy" → Download both JSON files
4. Replace old JSON files in your project folder
5. Re-drag folder to Vercel → Live in 30 seconds!
